Account-wide preferences and per-browser interface tuning.

## Per-browser preferences

Stored on this browser only. Don't affect other devices.

### Animations and visual effects

Turn off on slow devices. Disables decorative motion, smooth scrolling, blur, and glow effects. UI remains functional.

Two states surfaced:

- **Full visual effects enabled** — default.
- **Reduced effects active on this device** — set when off.
