> Compact, deterministic list of every skill and MCP tool that ships with Fibe.
> Generated by scripts/build-llm-skills.mjs from skills/ and seed-skills/.
> Each entry has three lines:
>   1. <name>: <description>
>   2. web:   https://whats.fibe.gg/<subdir>/<file>.md — raw markdown an LLM agent can fetch.
>   3. local: ./<subdir>/<file>.md — same file at the same path in a clone of this repo.
>
> For the linked index of full docs pages, see /llms.txt.
> For the concatenated full content, see /llms-full.txt.
